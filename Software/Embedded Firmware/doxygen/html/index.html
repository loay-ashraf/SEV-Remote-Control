This program is implemented on ATmega16/32 running at 16 MHZ CPU clock, it basically receives commands from Android companion application via Bluetooth using the widely common HC-05 Bluetooth Module, it acts upon the command received by -for example- turning ON a relay which activates certain feature of the vehicle; means of feedback include battery voltage and speed of the motor.

Pin Assignments
  • PB0-7: DAC output signal
  • PC0-7, PD6-7: Relay output signal
  • PD0: USART RX signal
  • PD1: USART TX signal
